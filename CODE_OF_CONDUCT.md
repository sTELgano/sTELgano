# Code of Conduct

## Our standard

sTELgano is a technical project with a serious purpose. We expect contributors
to be direct, evidence-based, and constructive. Disagreements about protocol
design and security tradeoffs are healthy and expected.

We do not tolerate:

- Personal attacks or targeted harassment
- Dismissing valid security concerns for non-technical reasons
- Discriminatory language or behaviour of any kind
- Deliberately introducing security vulnerabilities

## Enforcement

Report conduct issues to **conduct@stelgano.com**. Reports are handled
confidentially. Violations may result in removal from the project.

## Scope

This applies in all project spaces: GitHub issues, pull requests, discussions,
and any other forums where the project is represented.
